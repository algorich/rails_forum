#rails_forum

##Usage

Add in you Gemfile:

    gem 'rails_forum', :git => 'git://github.com/algorich/rails_forum.git'

##How to contribute
1. Take an issue.
2. Fork the project.
3. Write your **tests** and your features.
4. Send a pull request.
